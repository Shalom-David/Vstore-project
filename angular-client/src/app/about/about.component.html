Vstore

what is Vstore?

Vstore is a virtual store demo project built in Angular, and is using an express API built with mongoDB using Mongoose

what does it do?

In this website you can browse, and shop for various products, place, view and cancel uncompleted orders

Features

  • This project allows the user to have a smooth experience browsing the shop and adding products to cart
  • The main page is where the products are shown, anyone can browse for them but login is required to add items to your cart and place orders
  • Search bar with auto-complete and categories select menu can be used to filter product names and categories alike
    *this also applies to admin panel
  • The payment page requies billing information, the information will be fetched from the currently logged in user credit cards are NOT stored in the database. The user will have to pick a delivery date, only 3 deliveries possible per day, dates that are full will be marked in red
  • Successfully placing an order will give the user an option to immediatly download the receipt or go back to the main page
    **receipts can be downloaded at a ny time from the "My Orders" page
  • Users can change their profile details at any time while logged in
    **when changing username or password the user will be required to login again
  • Admins can add products, edit products, view orders and cancel orders.
